# Reminder Sheet

- `f|F<char>` + `t|T<char>` - search in line, `;` forward, `,` back
- `_` first char in line
- `0` start of line
- `<leader>?` - find recent opened
- `<leader>gf` - git files
- `<leader>e` - open diags
- `gd` - goto def
- `gr` - goto references
- `gI` - goto Implementation
- `<leader>D` - type def
- `K` + `C-K` - Signatures
- `<leader>ds` + `<leader>ws` - doc + ws symbols
- `:Format` - format it
- `[d` + `]d` - go through diags
